### Pointer Variable

一个可执行程序.exe由 代码(machine code) 和 数据(data - variables)构成，每个变量占有一到多个Byte.

那么这些 data 的地址 就可以用 Pointer Variable(指针变量)


**指针Pointer** 就是地址,
<br>
**指针变量Pointer Variable**是只存储地址的变量


#### Pointer Variable Declaration 
在变量前面放星号：
```c
int *p;     /*points only to integers*/
float *q;   /*points only to floats*/
char *r;    /*points only to chars*/
```

#### 间接寻址\* 以及 取地址符 &

- 取地址符 &

```c
int i, *p;
p = &i;    /*p points to i*/
// similarly
int i, *p = &i;
```

- 间接寻址 \* 
得到访问地址存的值，可以想象成 & 的反向操作
```c
 j = *&i    // same as j = i
 j = *p     // j = i
```

不要把间接寻址符用于未初始化的指针变量，如果变量没有初始化，那么他的值是未定义的

### Pointer as Function Parameters

C语言中函数的参数通过值传递，也就是说作为函数输入变量本身的值并不会改变，这个特性有好有坏

Pointer提供了一种不再传递变量x的方式，而是采用&x，即指向x的指针。

比如说我们想做一个函数，输入一个浮点数，然后要把这个数的小数点前后的数字分别输出
```c
void decompose(float x, int *intPart, float *fracPart);
{
    *intPart = (int) x;
    *fracPart = x - *intPart;
}
// 调用
decompose(3.1425926, &i, &f)
/*i = 3; f = 0.1415926*/
```

*Find the min and max element in an array*

```c
void max_min(int a[], int n, int *max, int *min);
{
    int i;

    *max = *min = a[0];
    for (i = 1; i < n; i++) {
        if (a[i] >*max)
            *max = a[i];
        else
            *min = a[i];
    }
}
```
