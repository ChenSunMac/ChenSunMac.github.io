MongoDB（来自于英文单词“Humongous”，中文含义为“庞大”）,是一种文档导向的数据库管理系统，由C++撰写而成. You can think of it as a simple dictionary 

or easily reference to the .bib files in LaTeX projects.

Install MongoDB is very easy in Unix like systems with **brew** one can easily brew everything needed.


### Run MongoDB Local


MongoDB 默认在本地使用/data/db目录作为数据库目录

One can simply 
```sh
$ mkdir -p /data/db
```
Then we can run mongodb without specifying the path

```sh
$ mongod
```

为了和根目录区分开方便管理和查找，我们可以specify the path of the data directoory

```
$ mongod --datapath /User/path/to/the/directory
```

然后数据库就run起来啦，这以后我们就可以开着数据库去跑我们的nodejs部分的东西。
